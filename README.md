# 局域网加密手机点呼系统

一个基于局域网的加密号码点呼方案，支持网页端控制手机外呼，同时对手机号码进行加密处理，保护隐私信息。

## 系统功能

- 局域网内网页端控制多手机外呼

- 手机号码加密展示，避免信息泄露

- 支持呼叫状态监控、历史记录查询

- 手机端无额外损耗，直接物理外呼

## 核心优势

1.  **隐私保护**：网页端仅展示加密号码，避免真实手机号泄露

2.  **低成本**：手机直接物理外呼，无需额外硬件/运营商接口

3.  **易操作**：员工通过网页即可完成拨号、挂断等操作

4.  **无损耗**：手机端无额外资源占用，长期使用稳定性高

## 使用流程

1.  在网页端「添加数据」模块输入加密号码（每行一个）

2.  提交后在「数据列表」选择目标号码，点击「拨打」

3.  手机自动执行外呼，网页端同步显示呼叫状态

4.  可在「拨打历史记录」查看过往呼叫记录
